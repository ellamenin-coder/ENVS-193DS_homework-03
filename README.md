# ENVS-193DS Homework-03

Spring 2026

## General information

This repo provides examples of many different things! 

The first section includes fitting, diagnosing, and visualizing linear model 
predictions, as well as using Pearson's Correlation tests to analyze the 
relationship between a predictor variable and a response variable.

The second section includes some figure-making out of personal data, both 
figures created to show the relationship between two variables.

The third section discusses a plan for affective visualization out of personal
data and can be useful for thinking about planning an affective visualization.

The fourth section is a statistical analysis on a paper analyzing correlation
between trace metals found in water. Specifically, this section looks into
analyzing a visual component of the paper.

## Data and file information

This is a 'data' folder in which data files are stored.

There is a 'code' folder in which data for all linear models are stored.

There is a 'drawings_and_rough_visualizations' in which my sketches and rough
drafts are stored.
